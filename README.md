# 🚀 Social Media Analytics Platform

An advanced social media analytics platform that aggregates and analyzes posts from multiple platforms with AI-powered sentiment analysis, mood detection, and comprehensive filtering capabilities.

## ✨ Features

### 📊 Multi-Platform Aggregation
- **Twitter/X** - Real-time tweets and mentions
- **Reddit** - Discussion threads and comments
- **YouTube** - Video content and community posts
- **Google News** - News articles and press coverage

### 🤖 AI-Powered Analytics
- **Sentiment Analysis** - Advanced emotion detection with confidence scoring
- **Mood Classification** - Positive, Negative, Mixed, and Neutral mood detection
- **Emotion Mapping** - 6-dimensional emotion analysis (Joy, Trust, Surprise, Anger, Sadness, Fear)
- **Content Safety** - Toxicity and spam detection
- **Topic Extraction** - Automatic keyword and topic identification
- **Analysis Stages** - Not Checked, In Progress, and Completed status tracking

### 🔍 Advanced Filtering System
- **Smart Search** - Real-time content filtering across title, content, and authors
- **Date Range Filtering** - Today, This Week, This Month, Last 3 Months with custom ranges
- **Platform Filtering** - Multi-select platform targeting
- **Engagement Filtering** - Sort by Recent, Popular, or Engagement metrics
- **AI Status Filtering** - Filter by analysis completion status

### 🎨 Modern UI/UX
- **shadcn/ui Design System** - Professional, accessible component library
- **Dark/Light Mode** - Seamless theme switching
- **Responsive Design** - Optimized for mobile, tablet, and desktop
- **Grid/List Views** - Flexible content display options
- **Real-time Updates** - Live filtering and sorting

### 📈 Detailed Analytics Dashboard
- **Sentiment Scores** - Numerical sentiment analysis (-100% to +100%)
- **Emotion Breakdown** - Visual emotion intensity mapping
- **Safety Metrics** - Content toxicity and spam assessment
- **Engagement Analytics** - Platform-specific interaction metrics
- **Interactive Modals** - Deep-dive post analysis with tabbed interface

## 🛠️ Technology Stack

- **Next.js 16.1.6** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS v4** - Utility-first styling
- **shadcn/ui** - Modern component library
- **Radix UI** - Accessible primitive components
- **Lucide React** - Beautiful icon system
- **date-fns** - Advanced date manipulation
- **React Day Picker** - Calendar functionality

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd social-media-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to [http://localhost:3000](http://localhost:3000)
